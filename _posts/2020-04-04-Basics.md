Some important things to remember

- *H(f(X)) <= H(X)*
- *H(X,Y) <= H(X) + H(Y)*



**Conditional Entropy**
<img src = "https://latex.codecogs.com/gif.latex?H(Y|X)=-\sum_{i,j}P_X(x_i,y_j)\log\frac{P_X(x_i,y_j)}{P(x_i)}">

Some useful points
- *H(Y|X) = H(X,Y)-H(X) = H(X|Y) + H(Y)-H(X)*
- *H(Y|X) <= H(Y)*
- *H(Y,X) = H(X) + H(Y) - I(X;Y)*
- *H(Y|X) = H(Y)* in case of independence
